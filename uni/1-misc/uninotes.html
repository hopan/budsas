About Unicode

(see also: Vietnamese version)

This page is written in Vietnamese language with Times New Roman font (version 2.76 or later), of Unicode standard, supplied by Microsoft when installing Windows-98 SE, Windows Me, Windows 2000, Windows XP, and future versions of Windows. For older versions of Win-NT, Win-98 and 95, the font used to be freely available from Microsoft website dor downloading. However, Microsoft has discontinued this offer.

As at July 2003, the font is still available at website: http://vietunicode.sourceforge.net/

This Times New Roman font is one of the 3 basic Microsoft core fonts which contain extra international characters including the Vietnamese characters. You can download these 3 core fonts (Times32.exe, Arial32.exe, Courier32.exe) from the above website. The *.exe is a self-installed file, when executed, will replace the old font with the newer version of Unicode standard.

 

 Adjust your web browser, either IE or Netscape, as following:

1. For IE 4.x: View --> Fonts --> Universal Alphabet (UTF-8)

2. For IE 5.x, 6.x: View --> Encoding --> Unicode (UTF-8)

3. For Netscape:

(a) View --> Encoding (or Character Set) --> Unicode (UTF-8)

(b) Edit --> Preferences --> Appearance-Fonts --> Use document-specified fonts

 

Printing Problems: Generally, you can print the webpage directly from your browser. However, if your printer fails to print the document properly, try one of the following methods:

1. Click: File --> Print --> Properties --> Print Quality --> Send True Type as Bitmap (or "Print Text as Graphics"; or set "Graphic Mode as HP-GL/2")
(usually for HP Laserjet printers).

2. Save the HTML page in your hard disk. Then, import into a Word document and print.
(You might have to adjust the printer as above).

3. Run Netscape (version 4.7 or later), access to the webpage, and print within Netscape
(You might have to adjust the printer as above).

4. If all fails, you might consider to purchase a Unicode-compliant printer, such as the Epson Color Stylus series (it's inexpensive and the printing quality is quite good).
